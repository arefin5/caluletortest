# Arithmetic Calculator REST API

This project provides a simple calculator functionality (addition, subtraction, multiplication, division, square root, and random string generation) with a credit-based system.

## Tech Stack
- Java
- Spring Boot
- Spring Security
- JWT
- MySQL
